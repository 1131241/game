# 暗棋象棋 - Dark Chess Game

一個基於網頁的暗棋象棋遊戲實現，遵守您描述的特殊規則。

## 遊戲規則

### 棋盤與初始佈局
- 使用棋盤的一半區域（4×8=32格）
- 雙方32枚棋子混合後面朝下隨機放置，每格一枚，無空格

### 玩家行動
每回合只能執行以下其中一項：
1. **翻棋**：將己方未翻開的棋子翻面朝上
2. **移動**：將己方已翻開的棋子移動一格（上下左右）
3. **吃子**：將己方已翻開的棋子吃掉符合規則的對方已翻開的棋子

### 吃子規則
- **一般棋子**：只能吃相鄰格（上下左右一格）的對方棋子
- **包/炮**：必須恰好隔著一格（中間恰好有一格）才能吃掉目標棋子
  - 中間的格子可以是：己方或對方的棋子，翻開或未翻開皆可
- **特殊規則**：
  - 兵/卒可以吃將/帥
  - 將/帥不能吃兵/卒
  - 其他吃子遵守大子吃小子原則（車 ≥ 馬 ≥ 炮 ≥ 士/相 ≥ 兵）

### 勝負條件
- 吃掉對方的將/帥即獲勝
- 或使對方無合法棋可走（將被困且無法翻開新棋子防衛）也算勝

## 技術說明

這個遊戲是一個單一的 HTML 檔案，包含：
- HTML5 結構
- CSS3 樣式（響應式設計）
- 原生 JavaScript 遊戲邏輯

無需任何外部依賴或建置過程。

## 如何運行

1. 確保 `index.html` 檔案存在於瀏覽器可存取的位置
2. 用現代網頁瀏覽器（Chrome, Firefox, Safari, Edge 等）開啟 `index.html`
3. 開始遊戲！

## 開發說明

如果您想修改或擴展這個遊戲：

### 修改棋盤大小
在 `DarkChessGame` 建構子中修改 `this.boardSize`：
```javascript
this.boardSize = { rows: 4, cols: 8 }; // 目前為4x8=32格
```

### 修改棋子分佈
在 `ALL_PIECES` 陣列中調整各方棋子數量和類型。

### 調整樣式
在 `<style>` 標籤內修改 CSS 來改變遊戲外觀。
